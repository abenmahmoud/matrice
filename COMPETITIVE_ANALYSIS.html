04 — Espaces libres

Ce que personne ne fait encore

Les opportunités non exploitées dans le marché actuel — zones où Matrice Narrative peut dominer sans combat direct.

Marché francophone entier

Aucun des 6 concurrents n'a d'interface ou de prompts en français. 300 millions de francophones, 0 outil spécialisé. Le marché existe et attend.

Taille estimée : 300M+ locuteurs · France / Québec / Belgique / Maghreb / Afrique

OS créatif Roman + Cinéma + Série

Tous les concurrents se spécialisent (roman OU scénario OU série). Personne ne couvre les 3 dans un seul pipeline cohérent avec un projet unique.

Valeur clé : un auteur peut pivoter d'un roman à une adaptation sans changer d'outil

Standards professionnels français

Note d'intention CNC, dossier de pitch français, format Fountain adapté — personne ne couvre les spécificités du cinéma francophone et de ses instances (CNC, SACD, festivals).

Uniquement Matrice Narrative actuellement

Scoring narratif quantifié

L'analyse de cohérence (H.P.S.A.) avec scores et corrections prioritaires n'existe nulle part ailleurs. C'est une approche unique de "diagnostic narratif".

Différenciateur fort pour auteurs professionnels

Ateliers d'immersion cinématographique

Aucun outil ne propose l'arc de tension interactif, la chambre des atmosphères, le mode réalisateur, ou la constellation de personnages dans un même espace.

11 outils Immersion = avantage démontrable en démo

Multi-agents narratifs spécialisés

Aucun concurrent n'utilise des agents IA distincts par rôle (script doctor, showrunner, éditeur). C'est la prochaine évolution de la catégorie — être premier est critique.

Fenêtre d'opportunité : 12–18 mois avant que les grands entrent

05 — Synthèse stratégique

Positionnement recommandé (Dunford)

Pour les auteurs et cinéastes francophones qui veulent transformer une idée en œuvre structurée et professionnelle, Matrice Narrative est un OS créatif complet qui guide tout le cycle — roman, scénario, série — via un pipeline IA en 5 phases. Contrairement à Sudowrite ou Novelcrafter, Matrice Narrative couvre Roman + Cinéma + Série TV dans un seul outil, en français, avec les standards professionnels (CNC, SACD, festivals).

Ce que nous gagnons

  • Seul outil 100% français avec prompts IA en français natif — monopole de facto sur 300M de francophones
  • Seul outil à couvrir Roman + Cinéma + Série dans un seul pipeline cohérent
  • Scoring H.P.S.A. et corrections prioritaires = différenciateur quantifiable unique
  • Standards CNC / Note d'intention = porte d'entrée naturelle vers les professionnels
  • 11 outils Immersion = expérience démo irréplicable en 10 minutes

Risques à surveiller

  • Sudowrite et Novelcrafter pourraient ajouter le français (ils ont les ressources)
  • OpenAI pourrait lancer un outil spécialisé (ChatGPT Projects est un signal)
  • La génération de prose longue reste un point faible — les concurrents sont meilleurs sur ce point
  • Pas encore de collaboration multi-utilisateurs (writers' room)

La fenêtre d'opportunité

  • 12–18 mois avant que Sudowrite / Novelcrafter localisent en français
  • L'ajout de multi-agents avant les concurrents crée un fossé technique difficile à combler
  • Le marché francophone professionnel (CNC, production) n'a jamais eu d'outil dédié

06 — Feuille de route d'évolution

3 phases pour dominer la catégorie

Priorités classées par impact / effort. Phase 1 = renforcer les fondations. Phase 2 = puissance IA. Phase 3 = écosystème.

1
Phase 1 — 0 à 3 mois · Fondations

Solidifier & Compléter le pipeline existant

Éditeur de prose intégré (markdown rich-text) Sauvegarde auto + historique de versions Génération par chapitre (prose réelle, pas seulement plan) Système de Notes contextuelles par module Dashboard analytics du projet (mots écrits, modules complétés) Export PDF formaté pour impression Mode focus (écriture plein écran)

Objectif : combler le gap prose longue face à Sudowrite/Novelcrafter. Un auteur doit pouvoir écrire le roman entier dans Matrice Narrative.

2
Phase 2 — 3 à 9 mois · Intelligence IA avancée

Multi-agents spécialisés + Mémoire du projet

Agent Script Doctor (analyse scène par scène) Agent Showrunner (cohérence inter-épisodes) Agent Éditeur Littéraire (style, voix, rythme prose) Mémoire vectorielle du projet (RAG sur toute l'œuvre) Détection de contradictions automatique Suggestions proactives basées sur le contexte Skills enseignables (auteur peut uploader ses propres références) Chat avec un agent qui connaît tout le projet

C'est la grande différenciation. Les concurrents anglais travaillent sur des fonctions similaires — être premier sur le marché FR avec des agents narratifs spécialisés crée un fossé.

3
Phase 3 — 9 à 18 mois · Écosystème & Communauté

Plateforme, collaboration & marché

Writers' Room (collaboration temps réel sur un projet) Intégration Final Draft / Highland (import/export) API publique (plugins tiers) Bibliothèque de templates (genres, formats) Marketplace de skills communautaires Portail Producteurs (partage sécurisé dossier pitch) App mobile (lecture + annotations) Intégration CNC (dépôt direct)

L'objectif est de devenir l'infrastructure du cinéma francophone indépendant, pas juste un outil d'écriture.

07 — Architecture multi-agents

Architecture d'orchestration recommandée

  • Orchestrateur central — un agent maître décide quel agent spécialisé appeler selon la tâche demandée par l'auteur
  • Mémoire partagée (RAG) — tous les agents lisent la même base de connaissance vectorielle du projet (matrice, personnages, lois d'univers, chapitres)
  • Résultats persistants — chaque rapport d'agent est sauvegardé et devient contexte pour les analyses suivantes
  • Mode pipeline — lancer tous les agents en séquence sur un projet = rapport de santé narratif complet en 1 clic

09 — Actions prioritaires

Les 5 choses à faire maintenant

1
Ajouter un éditeur de prose riche dans l'Atelier Roman
C'est le seul gap fonctionnel face à Sudowrite et Novelcrafter. Sans génération de prose longue, Matrice Narrative reste un outil de planification. Avec, c'est un concurrent direct.
2
Construire le skill "Standards CNC & SACD"
Injecter les critères CNC dans la Note d'Intention et le Dossier Pitch. Cela crée une valeur immédiate pour les cinéastes professionnels — un argument de vente que les concurrents anglophones ne peuvent pas répliquer.
3
Implémenter l'Agent Script Doctor
Premier agent spécialisé visible par l'utilisateur. L'analyse scène par scène avec HPSA local et suggestions de réécriture est un différenciateur immédiatement démontrables face à la concurrence.
4
Ajouter la mémoire vectorielle du projet (RAG)
Indexer tout le contenu du projet (matrice, personnages, chapitres, lois) dans un vecteur store. Les agents pourront ensuite interroger "tout ce que l'auteur a écrit" pour des réponses contextuelles et une détection de contradictions automatique.
5
Lancer sur Product Hunt France + réseaux scénaristes
La Fémis, La Scénaristerie, Unifrance, groupes Facebook/Discord de scénaristes francophones. La fenêtre d'opportunité (0 concurrent FR) justifie une prise de marché agressive maintenant, avant que Sudowrite localise en français.
