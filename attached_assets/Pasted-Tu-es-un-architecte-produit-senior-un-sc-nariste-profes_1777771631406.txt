TRÈS IMPORTANT : CORRIGER LE SENS DE H.P.S.A.
Actuellement H.P.S.A. semble défini comme Heroes / Plausibilité / Stakes / Arc.
Je veux changer complètement cela.

H.P.S.A. doit devenir :
- H = Humour
- P = Pleur / Puissance émotionnelle
- S = Suspense
- A = Attractivité

Ce score doit mesurer la capacité d’une histoire à :
1. faire rire sans casser l’émotion ;
2. faire pleurer ou toucher profondément ;
3. créer attente, mystère, tension et envie de continuer ;
4. donner envie immédiatement par son concept, son image, son personnage et sa promesse.

L’humour doit être organique, jamais parachuté.
Il doit venir :
- de la blessure du personnage ;
- de son masque social ;
- de sa contradiction ;
- de la situation ;
- du contraste entre quotidien et étrange ;
- d’un mécanisme de défense ;
- de l’enfant intérieur.

Le pleur doit venir :
- d’une vérité enfin dite ;
- d’une reconnaissance de la blessure ;
- d’une perte acceptée ;
- d’un pardon impossible ou tardif ;
- d’une phrase simple après beaucoup de silence ;
- d’un enfant intérieur enfin vu.

Le suspense doit venir :
- d’un secret ;
- d’un danger émotionnel ;
- d’une menace visible ou invisible ;
- d’une information retenue ;
- d’une question dramatique forte ;
- d’un retournement ;
- d’un conflit entre ce que le personnage veut et ce dont il a besoin.

L’attractivité doit venir :
- d’un concept clair ;
- d’une image forte ;
- d’un titre puissant ;
- d’un personnage mémorable ;
- d’un monde intrigant ;
- d’une promesse émotionnelle ;
- d’une originalité immédiatement perceptible.

IMPORTANT :
Le film ne doit pas être traité comme un roman découpé.
Le moteur doit convertir l’intérieur du personnage en cinéma visible.

Créer une fonction centrale :
CONVERTIR UNE ÉMOTION INTERNE EN SCÈNE FILMABLE

Exemple :
Entrée :
“Elle a peur de ne pas être entendue.”

Sortie :
- action visible : elle parle mais personne ne réagit ;
- objet symbolique : un verre d’eau qui vibre ;
- silence : le son disparaît au moment où elle essaie de dire la vérité ;
- conflit : son interlocuteur croit qu’elle exagère ;
- image forte : son reflet dans le miroir arrive avec une seconde de retard ;
- dialogue minimal :
  “Tu m’entends ?”
  “Tu n’as rien dit.”
- fin de scène : elle comprend que le problème n’est pas qu’on ne l’écoute pas, mais qu’elle-même ne sait plus reconnaître sa propre voix.
