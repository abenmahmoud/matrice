Create a professional full-stack MVP called “Matrice Narrative”.

IMPORTANT — LANGUAGE REQUIREMENT
The entire user-facing application must be in French.
All page titles, buttons, labels, forms, menus, dashboards, generated content, exports, placeholders, empty states and error messages must be in French.
Internal code, variable names, API routes and file names can remain in English for developer clarity, but the product experience must be 100% French.

Do NOT use the word “Bible”. Use “Matrice Narrative” everywhere.
The Matrice Narrative is the central source of truth for the fictional universe.

PRODUCT VISION
This is not a basic AI writing app.
It is a futuristic French creative studio that transforms a raw idea, a dream, a voice note, a memory, a dialogue or a story fragment into:
1. a structured fictional universe,
2. an emotional core,
3. a professional book structure,
4. a film screenplay structure,
5. a series structure,
6. a pitch document,
7. exportable creative documents.

CORE WORKFLOW
Idée brute → Matrice Narrative → Noyau Émotionnel → Recherche Vivante → H.P.S.A. → Livre → Scénario → Série → Pitch → Exports

TECH STACK
- React + Vite
- Tailwind CSS
- Node.js + Express
- SQLite for MVP
- Clean modular architecture
- No paid external API required for the first version
- Use high-quality deterministic mock generation now
- Prepare the architecture for future AI providers: OpenAI, Claude, Mistral, local models

DESIGN REQUIREMENTS
Create a very modern, premium, futuristic French interface.
Dark mode by default.
Elegant gradients, glassmorphism cards, soft glow, cinematic feeling, minimal typography.
Responsive desktop and mobile.
The product must feel like a professional creative OS for authors, filmmakers, artists and creators.
Do not make a toy app.
